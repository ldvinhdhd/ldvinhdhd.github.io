Hướng dẫn giả lập tay cầm chơi game

November 08, 2019


Giới thiệu:

Phần mềm Xbox 360 Controller Emulator dùng để giả lập tay cầm chơi game Xbox 360. Tức là biến các tay cầm của các hãng khác thành tay cầm Xbox 360 để chơi các game hỗ trợ tay cầm trên PC.

Tại sao lại giả lập tay cầm Xbox 360 mà không phải giả lập tay PS3, PS4...?

Đó là vì các game trên PC (Windows) chơi được bằng tay cầm thì đều phải hỗ trợ tay cầm Xbox 360, nhưng chưa chắc game đấy đã hỗ trợ tay PS hay các hãng khác.

Trên thị trường hiện này có 1 số tay cầm fake xbox 360, các tay cầm này cũng nên sử dụng kèm giả lập vì nó vẫn có đôi chút khác biệt so với tay cầm chính hãng, khi vào game bấm có thể bị sai phím.

Chỉ có tay cầm Xbox chính hãng mới không cần giả lập này thôi.

Hướng dẫn:

1. Tải phần mềm:

- Đối với game 32-bit: Tải x360ce.zip

- Đối với game 64-bit: Tải x360ce_x64.zip

Thực ra bình thường không thể xem ngay được 1 game là 32-bit hay 64-bit nên cứ tải cả 2 file trên về. Khi mở file .exe ra nó sẽ báo là có phù hợp không.

2. Giải nén file vừa tải về. Chúng ta được 1 file x360ce.exe (hoặc x360ce_x64.exe).

3. Copy file x360ce.exe vào cùng thư mục chứa file chạy game (file thực thi đuôi .exe).

4. Cắm tất cả tay cầm muốn dùng vào máy tính.

5. Chuột phải vào file x360ce.exe chọn Run As Admin.

6. Tiến hành cài đặt các nút bấm.

7. Lưu lại và tắt phần mềm.

Bây giờ có thể vào game chơi rồi!
